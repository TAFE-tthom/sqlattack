## Smallest Payment

Using the `payment` table, find the smallest amount paid where the amount is > 0. Output the amount and datetime it was processed.

* Refer: [SQLite Aggregate Functions](https://sqlite.org/lang_aggfunc.html)

---

You have access to the following table:

* `payment`
  * `payment_id`, `INT`
  * `customer_id`, `INT`
  * `staff_id`, `INT`
  * `rental_id`, `INT`
  * `amount`, `INT`
  * `payment_date`, `DATETIME`
  
Make sure you wait for the database to finish loading before running a query.
