## Counting Actors

You have been tasked with counting all the actors
that have the first name of **Paula**.

Use the an aggregate function to achieve this.

* Refer: [SQLite Aggregate Functions](https://sqlite.org/lang_aggfunc.html)

---

You have access to the following tables:

* `actor`
  * `actor_id`, `INT`
  * `first_name`, `VARCHAR`
  * `last_name`, `VARCHAR`

Make sure you wait for the database to finish loading before running a query.
