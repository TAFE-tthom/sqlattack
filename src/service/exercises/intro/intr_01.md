
## Hello World

Welcome to SQL! If you have never used SQL before, we will begin to learn it through a variety of exercises.

Your first task is to construct query to output `Hello World` using a `SELECT` statement.

Encapsulate text within `''` like so:

```
'Hello'
```

You can then specify the data after `SELECT`.

The expected output should be:

```
Hello World
```
