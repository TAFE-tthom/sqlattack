
## Staff and Customers

You will need to construct a query that will output the first name, last name and email of both customers and staff members.
The query will need to use a `UNION` and `alias` to construct this query. The output will be ordered by the first name in ascending order.

---

### Reference

* [Union Virtual Table](https://sqlite.org/unionvtab.html)
* [SQLite SELECT](https://sqlite.org/draft/lang_select.html)

---

Refer to the following information regarding the tables.

* `customer`
  * `customer_id`, `INT`
  * `store_id`, `INT`
  * `first_name`, `VARCHAR`
  * `last_name`, `VARCHAR`
  * `email`, `VARCHAR`
  * `address_id`, `INT`
  * `active`, `BOOLEAN`

* `staff`
  * `staff_id`, `INT`
  * `first_name`, `VARCHAR`
  * `last_name`, `VARCHAR`
  * `address_id`, `INT`
  * `picture`, `BLOB`
  * `email`, `VARCHAR`
  * `store_id`, `INT`
  * `active`, `BOOLEAN`
  * `username`, `VARCHAR`
  * `password`, `VARCHAR`

Make sure you wait for the database to finish loading before running a query.
