## Select - Actors - 1

Retrieve all the actors from the `Sakila` database.

Make sure you only output their `first_name` and `last_name`.

---

You have access to the following tables:

* `actor`
  * `actor_id`, `INT`
  * `first_name`, `VARCHAR`
  * `last_name`, `VARCHAR`

Make sure you wait for the database to finish loading before running a query.
