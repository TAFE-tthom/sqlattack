## Outputting Length of Names

You have been tasked finding out the length of each the last name for each actor. You will need to output their last name and the length of their last name.

You will also need to order by descending order.

---

### Reference

* [SQLite Functions](https://sqlite.org/lang_corefunc.html)

---

You have access to the following tables:

* `actor`
  * `actor_id`, `INT`
  * `first_name`, `VARCHAR`
  * `last_name`, `VARCHAR`

Make sure you wait for the database to finish loading before running a query.
