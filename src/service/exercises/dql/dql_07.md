## Select - Customers at Store 2 - 7

Retrieve all customers who were registered with a store of id `2`. Make sure only the `first_name` and
  `last_name` are outputted.

---

The database in use is `Sakila`.

You have access to the following tables:

* `customer`
  * `customer_id`, int
  * `store_id`, int
  * `first_name`, varchar/text
  * `last_name`, varchar/text
  * `email`, varchar/text
  * `address_id`, int
  * `active`, boolean
  
Make sure you wait for the database to finish loading before running a query.
