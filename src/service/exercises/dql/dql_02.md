## Select - Films - 2


Query all films that have a rating of `PG`. Make sure you output only the `id`, `title` and `release_year`.

---

The database in use is `Sakila`.


You have access to the following tables:

* `film`
  * `film_id`, `INT`
  * `title`, `TEXT`
  * `description`, `TEXT`
  * `release_year`, `INT`
  * `language_id`, `INT`
  * `original_language_id`, `INT`
  * `rental_duration`, `INT`
  * `rental_rate`, `REAL`
  * `length`, `INT`
  * `replacement_cost`, `REAL`
  * `rating`, `CHAR`
  * `special_features`, `TEXT`

Make sure you wait for the database to finish loading before running a query.
